Requirements of the Project:

Python Version 3.4 or above and DB Browser (SQLite) for database

About Project:

The School Management System is a simple project that contains all the necessary functionality for organizing the 
Student information inside a campus and contains:

-Login page.
-Student Profile.
-Fee report.

Developers: Nihal Gohil, Krupesh Kadakia and Avi Gupta.

How to run the Project:

1. Open file via Python IDLE.
2. Run index.py file.
3. The Login page will be open->Fill the login details.
4. After that Menu.py page will get open with Student Profile and Fee Report view option.
5. Click on View if you want to fill the Student Profile or Fee Report.
